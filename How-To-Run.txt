To run the programs, follow these steps:

1. Open the command prompt.
2. Navigate to the directory where the program is located.
3. Run the program using the command:

# Python
python py.py

# JavaScript
node js.js

# Java
java java.java

# C (Compile then Run)
gcc c.c -o c
.\c.exe

# C++ (Compile then Run)
g++ cpp.cpp -o cpp
.\cpp.exe

# C# (Compile then Run)
csc cs.cs
.\cs.exe
